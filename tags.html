Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1 – 6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div div blocks out elements and acts as a wrapper for styling
blockquote blockquote sections out a portion of text quoted from another source
header header container for header elements or introductory content
ol, ul, li ordered list, unordered list, list element creates ordered and unordered lists, and bullet points for list elements, respectively

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophisitication
strong strong make text appear bold <strong>at least a billion people</strong> at least a billion people
a anchor make a hyperlink <a href="http://learnenough.com">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code include code snippets <code>table</code> table
span span contains a portion of inline text and the <span>universe began to crumble upon itself</span> and we died and the universe began to crumble upon itself and we died

Document Tags

Tag Name Purpose
html HTML Tag for the entire HTML document.
head Head Tag for header data, titles and meta-data, etc. not appearing on the webpage.
body Body Includes all visible information seen in the HTML webpage.
title Title The title of the HTML document, seen in the tabs or windows of browsers.
meta Metadata Contains webpage metadata. Important for search engines and record-keeping.
